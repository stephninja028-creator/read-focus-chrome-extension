Read Focus Extension Test Page

Testing Instructions:

  1. Make sure the Read Focus extension is installed and enabled
  2. Click the extension icon in your browser toolbar
  3. Click "进入专注模式" (Enter Focus Mode)
  4. Test the highlighting feature by selecting text and using Ctrl/Cmd + H
  5. Test the notes feature by clicking the "摘录" (Notes) button
  6. Test the export feature by clicking "导出" (Export)
  7. Press Escape to exit focus mode
